# chatbot_for_depression
本科毕设，面向高校学生抑郁心理的聊天机器人
# 介绍
1. 闲聊对话多采用规则匹配，基于RNN的Seq2Seq模型;
2. 以Android App形式实现;
3. 面部情绪识别模型借用其他项目;
4. 闲聊对话内容较少;
5. 心理咨询模块参考Woebot,试图通过所谓的行为疗法，对用户进行鼓励，并对心情记录;
6. 记录用户每日心情;
7. 根据用户心情可以提供“讲个笑话”功能。
8. 添加一定的emoji，增添乐趣性。
